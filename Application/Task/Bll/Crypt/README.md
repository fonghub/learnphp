### 数据加密类型
- 单项hash加密：是指通过对不同输入长度的信息进行散列计算，得到固定长度的输出；加密是单向的，即不能对固定长度的输出进行计算从而获得输入信息。常用的hash加密算法有：
    * md5
    * sha1
